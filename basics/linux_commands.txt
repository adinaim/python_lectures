ctrl + alt + t  - открыть терминал на линуксе

ls - содержимое текущей директории
ls путь_до_директории - просмотр содержимого указанной директории
ls -a - просмтор скрытых файлов директории

mkdir название_директории - создание директории

cd путь_до_директории - смена директории

"." - ссылка на текущую директорию
".." - ссылка на директорию выше/родительскую
"~" - ссылка на домашнюю директорию

pwd - вывод абсолютного пути в консоль

абсолютный путь - путь, начинающийся с корневой/домашней директории (home/user/some_dir)
относительный путь - путь отоситльно други директорий (./some_dir)


tree

rmdir путь_до_директории - удаление пустой директории
rm -rf - принудительное удаление директории с содержимым

touch - создание файлов в текущей директории
touch название_директории - создание файла в другой директории

rm имя_файла - удаление файлов
rm -rf имя_файла - принудительное удаление чувствительных файлов
 
cat имя_файла - вывод содержимого файла в терминал
cat имя_файла >> имя_другого_файла - дозапись содержимого одного файла в другой

echo "какой-то текст" - вывод укаанного текста в терминал
echo "какой-то текст" >> название_файла - дозапись указанного теста в файл

mv названиие_файла новое_название_файла - переименование
mv название_файла путь_до_директории - перенос файла в указанную директорию

nano путь_до_файла - открытие файла с помощбю текстового редактора nano

Указанного файла не существует, он создает его 
первая_команда && последующая_команда - объединение команд
